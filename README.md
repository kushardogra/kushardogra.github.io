## âœï¸ Customization

### Update Personal Info
Edit `index.html` and replace:
- Name and contact details
- About me section
- Project descriptions
- Work experience
- Education details

### Add Profile Photo
Replace the placeholder image URL in the About section:
```html
<img src="./assets/profile.jpg" alt="Your Name">
```

### Add Resume
Place your resume PDF in the root folder as `Kushar_Dogra_Resume.pdf` or update the link in the hero section.

### Change Colors
Edit the CSS variables at the top of the `<style>` section:
```css
:root {
    --accent-cyan: #00d4ff;
    --accent-green: #00ff88;
    --accent-purple: #a855f7;
}
```
